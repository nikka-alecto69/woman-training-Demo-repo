# 女训 MVP 0.2

一个纯前端、移动端优先的「女性今日训练决策助手」原型。它不会替你做医学判断；它会把当日状态、训练基础、训练场景、目标与可选的周期信息，组合成更贴近今天的训练选择。

## 本版本整合内容

- 保留女训 0.2 的训练水平、训练场景、动作说明、训练/身体记录、真实 B 站视频链接包与本机管理演示功能。
- 视频链接包已合入：视频资源按「慢一点、练力气、出点汗」分类；点击「点开即练」或「去合集挑」会在新标签页打开相应的 B 站视频或合集。
- **新增独立「周期」页 + 训练库 v2（均移植自 jing-dev）**，底部导航现为「今日 / 周期 / 训练库 / 视频 / 记录 / 隐私」。

### 周期模块（本次合入）

- Apple Health 风格的整月日历：点按切换经期日，按住拖动可连续选择，切换月份后继续选择即可覆盖跨月经期。
- 自动把连续选中的日期合并为经期区间 `{start_date, end_date}`，并基于最近最多 3 次月经开始日建立周期模型：平均周期长度、平均经期长度、当前周期天数与所处阶段。
- 四阶段模型：月经期 / 卵泡期 / 排卵期 / 黄体期，配实时分析卡、三项指标卡、四阶段示意图（`assets/images/cycle_ring_v2.png`）与已识别经期区间列表。
- 训练推荐**优先采用日历周期模型**：当存在经期记录时按日历建模推算阶段；没有记录时自动回退到原有的「周期信息（可选）」填写字段，行为与女训 0.2 一致。
- 经期数据保存在新的 `nvxun_menstruationDays` 键中，仅存于当前浏览器；周期结果为概率推断（常见误差约 ±2-5 天），不作医学诊断。
- 本版本同时合入了 **周期模块** 与 **训练库 v2**（均来自 jing-dev）；视频资源仍沿用女训 0.2 的真实 B 站链接包。

> 结果页的周期提示卡仍使用根目录的 `cycle_ring_v2.png`；周期页的四阶段示意图使用 `assets/images/cycle_ring_v2.png`。

### 训练库 v2（本次合入）

- 在保留女训 0.2 原 13 套模板的基础上，新增 6 套器械/进阶训练：哑铃下肢力量、杠铃全身力量、器械臀腿塑形、绳索上肢线条、25 分钟低冲击 HIIT、普拉提核心稳定。
- 训练项升级为结构化字段：训练类型（Bodyweight / Dumbbell / Barbell / Machine / Cable / HIIT / Mobility-Recovery / Pilates-Yoga）、目标肌群、强度等级、时长范围、所需器械、动作演示占位图（内联 SVG，离线可用）。
- 训练卡与训练详情页展示演示占位图、类型、目标肌群、器械与时长范围；「训练库」页升级为 TRAINING LIBRARY V2 分层展示。
- 推荐引擎升级：综合**周期阶段 + 当日状态（冲刺/常规/降载/恢复）+ 训练水平 + 训练场景/器械 + 目标**筛选与排序训练（卵泡期/排卵期偏向负重训练，黄体期/经期与恢复日偏向低强度与活动度，减脂目标优先 HIIT）。
- 视频资源仍沿用女训 0.2 的真实 B 站链接包；v2 训练项保留 `videoCategories`，视频匹配照常工作。

> **本版本修正了 jing-dev v2 的两处分类行为：**
> 1. 训练类型推断：「无器械」模板不再被误判为 `Machine`（原因是「无器械」字符串含「器械」），现仅当器械为「健身房器械」时归类为 Machine，居家徒手模板正确归类为 `Bodyweight`。
> 2. 训练场景选择「都可以」时，推荐会从全部训练类型（含器械）中筛选，而不再只限徒手 / 活动度 / 普拉提 / HIIT。

## 运行方式

无需安装、无需后端：直接用浏览器打开 `index.html` 即可。所有数据只保存在当前浏览器的 `localStorage` 中，清除站点数据或换设备后不会保留。

也可以将 `v0.2nvxun` 目录直接作为 GitHub Pages 的发布目录。

## 使用说明

1. 首次打开后填写基础训练信息；属于有规律月经相关阶段的用户可选填周期信息。
2. 进入「周期」页，在日历上标记经期日期；系统会自动合并经期区间、推算阶段，并把结果用于当天训练建议。
3. 完成当天状态打卡后，查看训练建议、预计周期阶段和匹配的视频资源。
4. 完成训练后可填写反馈；规律力量训练可额外记录重量、次数、组数、RPE 与时长。
5. 点击底部导航的「视频」可浏览全部已整理资源；管理演示入口为 `#/admin`，演示密码为 `nvxun2026`。

## 数据与隐私边界

- 不登录、不上传、不接广告 SDK、不接 AI API，也不会将身体状态用于广告推荐。
- 视频链接只在用户点击后于新标签页打开；本页面不嵌入或抓取第三方视频。
- 周期阶段使用「预计」等表述，仅供训练参考，不能用于诊断、治疗、激素检测或替代医疗意见。
- 出现明显疼痛、异常出血、长期月经紊乱或严重不适时，应优先休息并咨询医生。
