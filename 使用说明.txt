女训 NüFit v1.0 · 试用说明
================================

这是一个纯前端、无需安装、无需联网的本地网页 App。
所有数据只保存在你自己的浏览器里，不会上传。

【最简单的打开方式】
1. 解压本压缩包，保持文件夹结构不变（index.html 与 src 文件夹要在一起）。
2. 双击 index.html，用浏览器（推荐 Chrome / Edge / Safari）打开即可使用。

【如果双击后页面空白】
个别浏览器对本地文件有安全限制，改用本地小服务器打开：
  - 装有 Python 3 的电脑：在本文件夹里打开终端/命令行，运行
        python3 -m http.server 8000
    然后浏览器访问  http://localhost:8000
  - 或用任意「静态网页服务器」工具指向本文件夹。

【说明】
- 数据保存在浏览器本机，换浏览器或清缓存会清空，属正常现象。
- 周期推算、训练与饮食建议仅供参考，不构成医疗诊断或处方。
